# 35 - 스프레드 문법

:  `...`은 하나로 뭉쳐 있는 여러 값들의 집합을 펼쳐서(전개, 분산하여, spread)개별적인 값들의 목록으로 만든다.

```jsx
console.log(...[1,2,3])
// 1 2 3
console.log(...'hello')
//h e l l o
```

스프레드 문법의 결과물은 값으로 사용할 수 없고, 쉼표로 구분한 값의 목록을 사용하는 문맥에서만 사용할 수 있다.
